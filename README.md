# CodingNepal Login Page

A modern and responsive login page created using HTML and CSS.  
This project contains a stylish navigation bar, background image, login form, and user-friendly design inspired by CodingNepal UI layouts.

## Features

- Responsive Login Page
- Modern UI Design
- Navigation Bar
- Email and Password Input Fields
- Login Button
- Signup Link
- User Icon Support
- Clean HTML and CSS Structure

## Technologies Used

- HTML5
- CSS3
